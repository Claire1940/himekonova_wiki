<Section icon={Target} title="姫子 ノヴァの最適なステータスと優先度">

完璧な**姫子 ノヴァのステータス**を実現するには、彼女の強力な自己バフと基本要求ステータスのバランスを取る必要があります。姫子 ノヴァはアクティブな状態に応じて最大200%の会心ダメージを自身に付与できるため、装備で最優先すべきは**会心率**と**攻撃力%**です。サンデーや花火などの「行動順繰り上げ」サポートと併用する場合、速度の優先度は下がります。

| ステータス種類 | 推奨値 | 優先度 | 理由 |
| :--- | :--- | :--- | :--- |
| **会心率** | 80% - 90%（非戦闘時） | **高** | 必殺技を確実に会心ヒットさせるため、戦闘中に100%に達する必要がある。 |
| **攻撃力** | 3,500 - 4,000+ | **高** | 高いダメージ倍率が基礎攻撃力に大きく依存するため。 |
| **会心ダメージ** | 130% - 150%（基礎） | **中** | 戦闘中の強力な100-200%バフがあるため、優先度は下がる。 |
| **速度** | 103（初期値）または135 | **低** | サンデーや花火による行動順繰り上げを使用しない場合のみ速度を盛る。 |
| **EP回復効率** | 100% - 119.4% | **中** | モチーフ武器（S1）やEP回復縄により、重い必殺技コストを緩和する。 |

<Callout type="warning" title="会心率の閾値">
会心率を軽視してはいけません。姫子 ノヴァの必殺技は多段ヒットするため、最後の一撃である「殲滅パルス（Annihilation Pulse）」で会心が発生しないと、どれだけ会心ダメージが高くても補いきれないほどの甚大なDPSロスが発生します。
</Callout>

</Section>

<Section icon={Swords} title="チームシナジーとローテーションガイド">

姫子 ノヴァのチームを編成するには、少なくとも1人の**「開拓の仲間」**が必要です。これらのキャラクターは、彼女を強化状態に移行させるための「トリガー役」となります。星魂0（無凸）では、「開拓の仲間」のみがアシストスキル使用後に追加ターンを獲得できるため、このメカニクスを最も効率的に活用できます。

**重要なチームの役割：**
- **トリガー役（エネイブラー）**：サンデー、丹恒-PT、またはRMC（「裁決」/「殲滅」を誘発）。
- **バッファー**：花火、ロビン、またはトリビー（行動順繰り上げやダメージアップ）。
- **耐久枠/サブDPS**：フォフォ（EP回復）、刃（ダブルDPS）、またはヴェルト（行動遅延）。

| 編成コンセプト | メンバー | 特徴・強み |
| :--- | :--- | :--- |
| **Hyper Verdict** | サンデー、花火、丹恒-PT | 最大限の単体バーストダメージ |
| **Dual State** | RMC, ヴェルト, 刃 | 混成ウェーブにおける高い汎用性 |
| **Sustainless** | RMC, ヴェルト, トリビー | エンドコンテンツ向けの超高速クリア |
| **Standard F2P** | 丹恒-PT, 三月なのか, 停雲 | 組みやすさと攻略のしやすさを両立 |

<Section icon={Trophy} title="星魂（凸効果）分析：凸2（2凸）は引くべき？">

姫子 ノヴァの星魂（凸効果）は非常に強力で、特に星魂1（1凸）と星魂2（2凸）は破格の性能です。星魂0（無凸）＋モチーフ武器（S1）でも完成されていますが、2凸することで戦闘の立ち回りが劇的にスムーズになり、扱いやすさが格段に向上します。

<CardGrid cols={3}>
  <Card title="星魂1（1凸）：速度と頻度">
    - 追加のアシストスキル使用回数+1。
    - 「裁決」/「殲滅」の誘発条件を緩和。
    - 必殺技の回転率向上に大きく貢献。
  </Card>
  <Card title="星魂2（2凸）：チーム編成の柔軟性">
    - 1ターンあたり**2回のアシストスキル**が使用可能。
    - **すべての味方**がアシスト使用後に追加ターンを獲得可能に。
    - ダメージ計算で希釈されない強力な1.3倍の乗算バフを獲得。
  </Card>
  <Card title="星魂6（完凸）：必殺技の超強化">
    - 「ソースエネルギー」の上限が6に増加。
    - アシストスキルがソースエネルギーを生成するようになる。
    - ビーム攻撃に160%の範囲（AoE）ダメージ倍率を追加。
  </Card>
</CardGrid>

<Callout type="tip" title="ガチャ・育成のアドバイス">
ガチャの引き数に限りがある場合、2凸（E2）はモチーフ武器（S1）確保よりも大きなダメージ上昇をもたらしますが、S1は立ち回りの快適性を大きく向上させます。ヘビーユーザー向けの「最もコスパが良い目標」は2凸1変（E2S1）です。
</Callout>
