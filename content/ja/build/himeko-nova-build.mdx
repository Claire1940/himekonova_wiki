<Section icon={Shield} title="光円錐の比較">

すべての光円錐が同じように作られているわけではありません。モチーフ光円錐が理想的ですが、無課金向けの代替選択肢もほとんどのコンテンツにおいて十分に実用的です。

| 光円錐 | ティア | 主なメリット |
| :--- | :--- | :--- |
| 夜を照らす星 | S+ | EP回復効率 + 防御力無視 |
| 今日も平和な一日 | A | 安定したダメージ上昇 |
| 朝食の儀式感 | B | 入手しやすい無課金向け選択肢 |
| 終わりなき演算 | B | 信頼性の高いステータス上昇 |

<Callout type="warning" title="EPに関する警告">
モチーフ以外の光円錐にはEP回復効率バフがありません。モチーフを使用しない場合は、EP回復効率の連結縄や、専用のEPバッテリーサポートで補う必要があります。
