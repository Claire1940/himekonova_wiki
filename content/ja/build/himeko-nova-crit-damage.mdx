<Section icon={Sparkles} title="最適な遺物と次元界オーナメント">

適切な遺物とオーナメントを選択することは、会心率を安定させる最も確実な方法です。姫子 ノヴァは自身で会心ダメージを大幅に強化するため、ステータスの希釈を避けるために、遺物では汎用的な与ダメージ倍率や攻撃力％を確保する必要があります。

4セット遺物は**「航海士は見ている」**が文句なしの最適解です。攻撃力12%の常時バフに加え、スキルと必殺技に最大54%の与ダメージ増幅を提供します。アシストスキルは通常のスキル使用としてカウントされるため、姫子 ノヴァはこのバフを常に最大スタックで維持できます。「ウェーブストライダー」や「学者」セットは一時的な代用にはなりますが、総ダメージ量では大きく劣ります。

| 遺物 / オーナメント | 2セット効果 | 4セット / セット効果 | 評価 |
| :--- | :--- | :--- | :--- |
| **航海士は見ている** | 攻撃力+12% | スキル＆必殺技の与ダメージ+54% | ★★★★★ |
| **ウェーブストライダー** | 与ダメージ+10% | 条件付きで必殺技ダメージ強化 | ★★★☆☆ |
| **堕ちた星 (オーナメント)** | 会心率+8% | ナナシの仲間編成時、会心ダメージ+32% | ★★★★★ |
| **バナナアミューズメント** | 会心率+8% | 会心ダメージ+36% | ★★★★☆ |
| **星々の競技場** | 会心率+8% | 会心率70%以上で通常/スキルダメージ+20% | ★★★★☆ |

次元界オーナメントについては、**「堕ちた星」**セットを強く推奨します。常時会心率+8%に加え、装備者と味方の少なくとも1人が「ナナシの仲間」派閥に属している場合、さらに会心ダメージ+32%を得られます。該当する仲間を編成しない場合は、制限なしでほぼ同等の価値を持つ**「バナナアミューズメント」**が選択肢に入ります。

<Callout type="warning" title="ステータス希釈の警告">
常時「殲滅」状態を維持するチームでは、200%もの会心ダメージバフによってステータスが希釈される可能性があります。このような特定の編成では、「星々の競技場」のような汎用的な与ダメージ％を提供するセットの方が、純粋な会心ダメージセットを上回ることがあります。
</Callout>

<Section icon={Target} title="ステータスの目安と装備設定">

姫子 ノヴァは自身のキットと味方から最大200%のボーナス会心ダメージを受け取るため、胴体に「会心ダメージ」を採用すると収穫逓減（効率低下）を招き、ダメージが不安定になりがちです。すべての攻撃を確実に会心させるため、胴体は「会心率」を選択することがほぼ必須となります。

速度の要求値はチーム構成に大きく依存します。サンデーや花火のような行動順を早めるサポーターと組む場合は、速度を捨てて（低速ビルド）攻撃ステータスに特化しましょう。行動順加速サポーターがいない場合は、ローテーションを維持するために速度134または135のブレイクポイントを目指すことを推奨します。

<Steps>
  <Step num="1" title="胴体を選択する">
    安定した会心のために「会心率」の胴体を装備しましょう。「会心ダメージ」への切り替えは、サブステータスが非常に優秀で、かつ記憶の開拓者のような強力な会心率バッファーがいる場合のみ検討してください。
  </Step>
  <Step num="2" title="脚部を選択する">
    サンデーや花火と組む場合は「攻撃力%」を選択します。伝統的なダブルアタッカー編成や低速ユーティリティ編成の場合は「速度」に変更してください。
  </Step>
  <Step num="3" title="オーブと連結縄を選択する">
    オーブは「攻撃力%」または「属性与ダメージ」を装備します。連結縄は「攻撃力%」が標準ですが、モチーフ光円錐がない（S0）場合は、重い必殺技コストを補うために「EP回復効率」も非常に有効です。
  </Step>
  <Step num="4" title="サブステータスの最適化">
    まずは戦闘中の会心率が90%～100%に達することを目指してください。達成後は、残りのリソースをすべて「攻撃力%」と「会心ダメージ」に注ぎ込みます。
  </Step>
</Steps>

| 装備スロット | メインステータス | 推奨サブステータス |
| :--- | :--- | :--- |
| **胴体** | 会心率 | 会心ダメージ, 攻撃力%, 速度 |
| **脚部** | 攻撃力% / 速度 | 会心率, 会心ダメージ, 攻撃力% |
| **オーブ** | 攻撃力% / 与ダメージ% | 会心率, 会心ダメージ, 速度 |
| **連結縄** | 攻撃力% / EP回復効率 | 会心率, 会心ダメージ, 攻撃力% |

<Callout type="success" title="戦闘ステータスの目標">
最高のパフォーマンスを発揮するために、非戦闘時のステータスで以下を目指しましょう：**攻撃力 3500+**、**会心率 80%**（戦闘中バフで容易に100%に到達）、**会心ダメージ 130% ～ 150%**。
</Callout>

</Section>

<Section icon={Trophy} title="光円錐ティアリスト">

姫子 ノヴァのモチーフ光円錐**「帆 (S1)」**は、知恵の運命において最も強力な選択肢の一つです。非常に高い基礎攻撃力を持ち、常時32%の防御無視を付与し、アシストスキルを発動するたびにEPを6回復します。さらに、アシストスキルと必殺技のダメージを永久に最大60%増加させるスタック型のバフも備えています。

モチーフを引かない場合、代わりの知恵光円錐はやや限られます。ショップで入手できるヘルタ光円錐が強力な無課金向けの選択肢となるほか、「銀河鉄道の夜」は虚構叙事コンテンツにおいて依然として優秀な選択肢です。

| 光円錐 | 防御無視 | 与ダメージバフ | ユーティリティ / EP | 評価 |
| :--- | :--- | :--- | :--- | :--- |
| **帆 (S1)** | **32% (常時)** | +60% (アシスト/必殺) | アシスト毎にEP+6 | ★★★★★ |
| **ヘルタ光円錐** | 0% | 中程度の汎用バフ | なし | ★★★★☆ |
| **銀河鉄道の夜** | 0% | 高い条件付きバフ | なし (虚構叙事限定) | ★★★☆☆ |
| **今日も平和な一日** | 0% | 安定したスケーリング | なし | ★★★☆☆ |

<Callout type="warning" title="ガチャ優先度のアドバイス">
彼女の星魂2（E2）は非常に強力で、どの味方でも追加ターンをトリガーでき、ターン毎にアシストチャージを2回復できるようになります。リソースがある場合は、モチーフ光円錐よりもE2を優先することを強くお勧めします。E1とS1で迷っている場合は、EPの快適性を求めてS1を引くのが良いでしょう。
