<Section icon={Shield} title="おすすめの姫子 ノヴァ遺物セット">

ハイブリッドな役割を担う姫子 ノヴァにとって、適切な**姫子 ノヴァの遺物セット**を選ぶことは非常に重要です。戦闘開始時に自動でスキルを発動し、必殺技によるスターブレイザー召喚に大きく依存するため、スキル使用やEP管理を強化するセットが優れています。現在のメタでは、ローテーションと完璧に噛み合うダメージバフを蓄積できる「ナビゲーター」セットが推奨されます。

| 遺物セット | 2セット効果 | 4セット効果 | 推奨理由 |
| :--- | :--- | :--- | :--- |
| **ナビゲーター** | 攻撃力+12% | スキル/必殺技ダメージ+18%（最大3層） | 汎用的な最適解 (BiS) |
| **荒野の盗人** | 会心ダメージ+16% | 味方からターゲット後、攻撃力+48% | ハイパーキャリー必殺技特化 |
| **溶岩で鍛造する火匠** | 炎属性ダメージ+10% | スキルダメージ+12% / 炎属性ダメージ+12% | 安定した無課金向け代替案 |
| **混合セット** | 炎属性ダメージ+10% | 攻撃力+12% | ゲーム序盤の周回用 |

「ナビゲーター」セットが優れている理由は、姫子 ノヴァが秘技のおかげで戦闘開始時からスキルがアクティブな状態であるためです。これにより、ほぼ即座に最大スタックに到達し、最も強力なアビリティに54%ものダメージバフを付与できます。一方で、「荒野の盗人」セットは、花火やサンデーのように頻繁にバフをかけて4セット効果を誘発できるサポーターと組む場合に非常に強力です。

<Section icon={Map} title="姫子 ノヴァの最強チーム編成">

姫子 ノヴァは、メインハイパーキャリーとしても強力なサブDPS兼サポーターとしても機能する柔軟なユニットです。「コンパニオン・プロトコル」により、チームのタイプに合わせて適応可能です。「評決」プロトコルは必殺技主体のチームに、「殲滅」プロトコルは攻撃的な高頻度攻撃チームに適しています。

**チーム1: バランス型プレッシャー**
- **メインDPS**: 姫子 ノヴァ
- **デバッファー**: モート・ブレイド（防御力ダウン）
- **サポーター**: 花火（会心ダメージ＆行動順加速）
- **耐久**: 丹恒・飲月（バリア＆ユーティリティ）

**チーム2: ハイパーキャリー特化**
- **メインDPS**: 姫子 ノヴァ
- **サポーター1**: サンデー（EP＆会心バフ）
- **サポーター2**: 花火（スキルポイント効率）
- **耐久**: 丹恒・飲月（保護）

| 味方キャラ | シナジーの役割 | 採用理由 |
| :--- | :--- | :--- |
| **花火** | 行動順加速 | 速度脚を捨てて攻撃力を盛れるようにする |
| **サンデー** | EPバッテリー | スターブレイザーの召喚頻度を向上 |
| **モート・ブレイド** | 防御力減少 | 姫子 ノヴァの高い基礎ダメージを乗算 |
| **丹恒・飲月** | バリア付与 | 「開拓」シナジーのための安全確保 |

<Callout type="tip" title="プロトコルの選択">
サンデーと組む場合は、必殺技ダメージを最大化するために**「コンパニオン・プロトコル：評決」**を使用してください。テンポの速いチームでは、より頻繁に無料アシストスキルを発動するために**「殲滅」**を使用しましょう。
