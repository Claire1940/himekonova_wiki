<Section icon={Sparkles} title="Prioridades de Estadísticas y Optimización">

Elegir las estadísticas principales correctas en tus reliquias es tan importante como el bono de set en sí. Para Himeko Nova, equilibrar su relación de CRIT es la mayor prioridad. Debido a que depende de habilidades de múltiples golpes, una Prob. CRIT baja puede resultar en números de daño extremadamente inconsistentes.

| Pieza de Reliquia | Estadística Principal | Razón |
| :--- | :--- | :--- |
| **Cuerpo** | **Prob. CRIT** | Asegura un daño consistente en múltiples golpes |
| **Botas** | **ATQ%** | Maximiza el escalado base para la Hab./Def. |
| **Esfera** | **Bono Daño Fuego** | Multiplicador esencial para DPS elemental |
| **Cuerda de Unión** | **Recarga de Energía** | Rotaciones de Definitiva más rápidas para mayor DPS |

**Prioridad de Sub-stats:**
1. **Prob. CRIT**: Apunta a al menos un 70% para consistencia.
2. **Daño CRIT**: Acumula tanto como sea posible una vez que la Prob. CRIT sea estable.
3. **Velocidad**: Útil para alcanzar los umbrales (134+), aunque el ATQ% suele ser competitivo.
4. **ATQ%**: Proporciona una base sólida para todos los cálculos de daño.

<Callout type="info" title="Nota de Build">
Aunque la Velocidad suele ser la reina en el combate por turnos, Himeko Nova se beneficia tanto del ATQ puro que solo deberías cambiar a botas de Velocidad si puedes alcanzar el umbral de 134 sin sacrificar demasiado valor de CRIT.
</Callout>
